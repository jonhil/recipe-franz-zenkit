# zenkit.com Franz Plugin
Recipe for [zenkit.com](https://www.zenkit.com) integration with [Franz](https://meetfranz.com)

## Requirements
This recipe only works for Franz 5.X, get it now at http://meetfranz.com

## How to test this recipe
1. To install a new integration, download the integration folder.
2. Open the Franz Plugins folder on your machine:
  * Mac: `~/Library/Application Support/Franz/recipes/dev/`
  * Windows: `%appdata%/Franz/recipes/dev/`
3. Copy the folder into the plugins directory
4. Reload Franz
